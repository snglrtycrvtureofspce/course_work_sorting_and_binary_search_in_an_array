RU
--------------------------------------------------------------------------------------------------------------------------------------------------------
Курсовая работа по объектно-ориентированному программированию, на языке программирования C++ с использованием графического интерфейса Windows Forms.
Тема курсовой работы - "Программа для сортировки и бинарного поиска в массиве"
--------------------------------------------------------------------------------------------------------------------------------------------------------
Возможности:
	* Авторизация и аутентификация пользователей.
	* Локализация на английском, русском и белорусском языках.
	* Возможность изменения фона (картинка, цвета, случайные цвета).
	* Справочная система приложения.
	* Ввод пользователем данных о размере массивов, ввод их содержимого и вохможность рандомного заполнения массивов.
	* Сортировка Хоара
	* Бинарный поиск в таблице символов
	* Визуальзация массивов и содержимое файлов, возможность чтения и запись файл (-ов) в форматах *.docx, *.txt и (или) *.bin.
--------------------------------------------------------------------------------------------------------------------------------------------------------
Правила пользования с основным окном программы:
1. Для начало укажите размер массива (указать можно: 
	а) самостоятельно вписав значения в поля "Размер массива:";
	б) нажать на кнопку "Рандом размер" и поля автоматически будут заполняться значениями при каждом нажатии кнопки
2. Далее нужно создать сам массив, сделать это можно двумя способами:
	а) нажать на кнопку "Рандом массива" и массив будет автоматически заполняться случайными значениями при каждом нажатии кнопки
	б) нажать на кнопку "Создание массива", но тогда массив придётся заполнять вручную, для этого существует кнопка "Добавление числа в массив",
	а также поле выше в которое требуется вводить значение по порядку
--------------------------------------------------------------------------------------------------------------------------------------------------------
Мини-ЧаВо:
В: Как зайти в окно программы?
О: Пройти авторизацию и аутентификацию пользователя.

В: Как сменить язык или оформление?
О: Нажать кнопку меню в правом верхнем углу экрана.

В: Программа выдаёт критическую ошибку при запуске!

Coursework on object-oriented programming, in the C++ programming language using the Windows Forms graphical interface.
Theme of the course work - "A program for sorting and binary search in an array"
-------------------------------------------------- -------------------------------------------------- --------------------------------------------------
Capabilities:
* Authorization and authentication of users.
* Localization in English, Russian and Belarusian.
* Ability to change the background (picture, colors, random colors).
* Application help system.
* User input of data on the size of arrays, input of their contents and the possibility of random filling of arrays.
* Hoare sort
* Binary search in symbol table
* Visualization of arrays and the contents of files, the ability to read and write file (s) in *.docx, *.txt and (or) *.bin formats.
-------------------------------------------------- -------------------------------------------------- --------------------------------------------------
Terms of use with the main window of the program:
1. First, specify the size of the array (you can specify:
a) by independently entering the values ​​in the fields "Array size:";
b) click on the "Random size" button and the fields will be automatically filled with values ​​each time the button is pressed
2. Next, you need to create the array itself, you can do this in two ways:
a) click on the "Random array" button and the array will be automatically filled with random values ​​each time the button is pressed
b) click on the "Create array" button, but then the array will have to be filled in manually, for this there is a button "Adding a number to the array",
as well as the field above in which you want to enter a value in order
-------------------------------------------------- -------------------------------------------------- --------------------------------------------------
Mini FAQ:
Q: How to enter the program window?
A: Pass authorization and user authentication.

Q: How can I change the language or design?
A: Press the menu button in the upper right corner of the screen.

Q: The program gives a critical error on startup!
A: Make sure you have .NET Framework v2.0 or later installed.

Курсавая праца па аб'ектна-арыентаваным праграмаванні, на мове праграмавання C++ з выкарыстаннем графічнага інтэрфейсу Windows Forms.
Тэма курсавой працы - "Праграма для сартавання і бінарнага пошуку ў масіве"
-------------------------------------------------- -------------------------------------------------- --------------------------------------------------
Магчымасці:
* Аўтарызацыя і аўтэнтыфікацыя карыстальнікаў.
* Лакалізацыя на англійскай, рускай і беларускай мовах.
* Магчымасць змены фону (карцінка, колеры, выпадковыя колеры).
* Даведачная сістэма прыкладання.
* Увод карыстальнікам дадзеных аб памеры масіваў, увод іх змесціва і пахабнасць рандомнага запаўнення масіваў.
* Сартаванне Хоара
* Бінарны пошук у табліцы сімвалаў
* Візуальзацыя масіваў і змесціва файлаў, магчымасць чытання і запіс файл (-аў) у фарматах *.docx, *.txt і (або) *.bin.
-------------------------------------------------- -------------------------------------------------- --------------------------------------------------
Правілы карыстання з асноўным акном праграмы:
1. Для пачатку пазначце памер масіва (паказаць можна:
а) самастойна упісаўшы значэнні ў палі "Памер масіва:";
б) націснуць на кнопку "Ранда памер" і палі аўтаматычна будуць запаўняцца значэннямі пры кожным націску кнопкі
2. Далей трэба стварыць сам масіў, зрабіць гэта можна двума спосабамі:
а) націснуць на кнопку "Рандом масіва" і масіў будзе аўтаматычна запаўняцца выпадковымі значэннямі пры кожным націску кнопкі
б) націснуць на кнопку "Стварэнне масіва", але тады масіў прыйдзецца запаўняць уручную, для гэтага існуе кнопка "Даданне ліку ў масіў",
а таксама поле вышэй у якое патрабуецца ўводзіць значэнне па парадку
